
# Easyspoon

Easyspoon - a __Express/Node + React__ application for meal and grocery planning.

Built with the [spoonacular_api](https://spoonacular.com/food-api)

[Spoonacular SDK](https://github.com/ddsky/spoonacular-api-clients/tree/master/javascript)

**in progress**

NOTE - to run this **(in progress)** app, clone Spoonacular SDK and place in this project's root directory in a folder named "spoonacularSDK".

Root Directory should look like this -

 - backend/
 - diagrams/
 - proposal/
 - **spoonacularSDK/**
 - .gitignore
